# Verificador-de-idade
Um Site que utiliza da interatividade do JavaScript segundo a entrada do usuário, detectando a idade e o gênero do usuário
Primeiramente o usuário deve inserir seu ano de nascimento, após isso, marcar a check-box referente ao seu gênero e 
por fim clicar no botão "Verificar" para que seja impresso no HTML sua idade e se o usuário é homem ou mulher
